# Abstracts Lie? Measuring Thematic Alignment Between Journal Scope and Article Abstracts

## Research Question

How reliably do article abstracts represent the thematic focus declared in a journal's Aims & Scope?

This project investigates whether article abstracts can be used as reliable proxies for the thematic focus of a scientific journal. The case study is the Journal of Machine Learning Research (JMLR).

The informal title "Abstracts Lie?" is used as a motivation. The project does not claim that abstracts are false. Instead, it measures how closely article abstracts align with the journal's official thematic description.

## Dataset

The project uses:

- JMLR Aims & Scope text from the official JMLR website;
- 80 article abstracts collected from recent JMLR volumes.
